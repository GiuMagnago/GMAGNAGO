Sobre mim

Olá, meu nome é Giuseppe Chaves Magnago, sou um freelancer de 16 anos que mora no Brasil, sou um estudante de Ciência da Computação na PUC Minas. Já desenvolvi alguns projetos de programação, tais como websites, bots, jogos, softwares entre outros projetos mais simples.

Tenho experiência com Front-End (2 anos), JavaScript (2 anos), Python (3 anos), C and C++ (1 ano). Criei este site para ser uma maneira mais simples de mostrar às pessoas sobre meus trabalhos, experiências, mais sobre quem eu sou e, pelo menos para mim, isso parece mais profissional do que apenas mostrar meus repositórios no GitHub ou perfil no LinkedIn.

Nasci em 2004, em Imperatriz-MA, estudei o ensino fundamental no Colégio Santa Cruz, em Araguaína-TO, e finalizei o ensino médio na Escola Santa Teresinha, em Imperatriz-MA. Tive o primeiro contato com programação aos meus 14 anos e me apaixonei, estudei python primeiramente, depois comecei a trabalhar com Web e aprendi HTML, CSS e JavaScript.
